somunia ( ソムニア ) とは、歌の投稿をメインに行なっているバーチャルYouTuberである。YouTubeのほか、各種音楽配信サイトでもオリジナル曲が配信されている。

概要

somunia
YouTube somunia
Twitter @she_is_wired
タグ #somunia
アート #somu_art

水色の髪・瞳をした少女の姿をしている。自己紹介動画やQ&A動画では何処かの部屋のベッドに腰かけて話しているが、「物心ついた時からこの部屋にいる」「部屋から出たことが無い」「少し前まではもっとちゃんと昔のことをおぼえていたが、時間が経つにつれて思い出せなくなってきた」というどこか不穏な情報が提供されている。「私が私であることを保てるように」動画を残しているらしい。

2018年7月31日にYouTubeに投稿された自己紹介動画以後、オリジナル曲やカバー楽曲の歌唱を中心として投稿・活動している。featuringなどのかたちで他のアーティストの楽曲に参加することもある。2019年5月3日現在、最も再生回数の多いオリジナル楽曲動画は「summer leap (short ver.)」、カバー楽曲動画は「［ cover ］波よせて - クラムボン」。

これらの楽曲は各種デジタルミュージックサイトで公開/頒布している（本記事「関連商品」や「関連リンク」を参照）。また、物理CDも同人イベント（音系同人即売会「M3」など）やBOOTHなどで頒布されている。

バーチャルIPプロダクション「Ficty」のプロデュースを受けているようで、同じFictyプロデュースのバーチャルYouTuber「KMNZ」のLITAとLIZを「先輩」と呼んでいる。
